## 🚀 Project Overview

💡 The **Healthcare CSV Agent** is an AI-powered data analysis tool that allows users to interact with healthcare datasets using **natural language queries**.

📊 Instead of writing complex SQL or Python code, simply **ask questions** and get instant insights!

---

## ✨ Features

✅ Natural Language Querying
✅ Automated Data Insights
✅ CSV File Processing
✅ AI-Powered Analysis
✅ Easy Integration with Python
✅ Interactive Data Exploration
